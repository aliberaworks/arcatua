はじめに — Arcatuaの使い方

AIツール集です。アプリによってAPIキーが必要なものと不要なものがあります。

1
アプリを選ぶ

一覧からアプリをクリックすると、このページ内で即起動します。「無料」タグのアプリはAPIキー不要です。

2
AIアプリを使う場合：APIキーを設定する

右上の「⚙ API設定」からAPIキーを入力してください。一度設定すれば全アプリで共通利用できます。

3
APIキーの取得方法（無料で始めるなら）

Gemini（Google）：aistudio.google.com で無料取得。カード登録不要。
Claude（Anthropic）：console.anthropic.com で取得。
完全無料・ローカル：Ollama をPCにインストールすれば、APIキー不要でAIが動きます。

4
キーはどこに保存される？

あなたのブラウザのlocalStorageのみ。外部サーバーには送信されません。

💡 アプリはどんどん追加されます。新着をフォローするには note か aliberaworks.com をチェック。

API / モデル設定

APIキーを一度設定すれば、全アプリで共通して使われます。キーはあなたのブラウザにのみ保存されます。
